# ai_and_automations_repository

Here you can find a personal collection of AI-powered automations and intelligent workflows — from smart bots to end-to-end pipelines, making machines think and act.

---

## What's in here?

This repo is a living collection of experiments and production-ready solutions at the intersection of **AI** and **process automation**. Each project lives in its own folder with its own documentation.

Projects are organized by category:

```
├── ai-agents/        # LLM-powered agents and bots
├── automations/      # Workflow automations (n8n, Power Automate, UiPath)
├── apps/             # Low-code apps (Power Apps)
├── pipelines/        # End-to-end data + AI pipelines
```
